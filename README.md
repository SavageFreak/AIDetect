# PuzzleAI

Αυτόματος λύτης του παζλ επαλήθευσης (NEWMT2). Διαβάζει την ελληνική οδηγία,
βρίσκει το σωστό εικονίδιο στο πλέγμα και το σέρνει στο κουτί επαλήθευσης.

## Εγκατάσταση (μία φορά)

1. Κατέβασε το **`PuzzleAI.zip`** από τα [Releases](../../releases/latest).
2. Κάνε **extract** τον φάκελο όπου θες.
3. Τρέξε **`start_puzzle_ai.bat`** (ή `PuzzleAI.exe`).

Δεν χρειάζεσαι τίποτα άλλο — Python, CUDA, Tesseract και το εκπαιδευμένο μοντέλο
είναι ήδη μέσα. GPU (AMD/NVIDIA/Intel) ενεργοποιείται αυτόματα μέσω DirectML.

## Αυτόματη ενημέρωση

Κάθε φορά που ανοίγεις την εφαρμογή, ελέγχει **μόνη της** για ενημέρωση και την
κατεβάζει στο παρασκήνιο (μικρό patch, όχι όλο το πρόγραμμα). **Δεν χρειάζεται να
ξανακατεβάσεις τίποτα** — απλά την ανοίγεις και είναι πάντα η τελευταία έκδοση.
